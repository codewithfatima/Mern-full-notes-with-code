### Variables, Data Types & Operators in C++
Variables

Variables are containers that store data. You must declare the type of variable before using it.
Example:

int age = 16;       // Integer variable
double height = 1.75; // Floating-point variable
char grade = 'A';   // Character variable

Data Types

Common C++ data types:

int → Integer numbers

double → Decimal numbers

char → Single character

string → Sequence of characters

bool → true/false

Operators

Operators perform operations on variables and values:

Arithmetic: +, -, *, /, %

Comparison: ==, !=, >, <, >=, <=

Logical: &&, ||, !


int a = 5, b = 3;
cout << a + b; // 8
cout << (a > b); // 1 (true)
